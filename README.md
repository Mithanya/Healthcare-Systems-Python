# Healthcare IT Systems Simulator in Python

Healthcare IT Systems Simulator is a collection of Python programs designed to simulate real-world hospital and biomedical workflows. The project models various healthcare processes such as patient admission, ICU monitoring, diagnostic record management, and medical resource tracking.

This project was developed as part of hands-on biomedical training to demonstrate how software systems can support hospital operations and improve efficiency in healthcare environments.

---

## Project Overview

The system consists of multiple Python scripts, each representing a specific hospital workflow. These programs simulate day-to-day operations within healthcare facilities and highlight how programming can be applied to healthcare technology and hospital information systems.

---

## Project Modules

| Day | Module | Description |
|-----|--------|-------------|
| 1 | Patient Management | Admit, view, and discharge patients |
| 2 | Medical Gas System | Request and track medical gases such as oxygen |
| 3 | OT Surgery Scheduler | Schedule and view surgery timings |
| 4 | ICU Vital Monitor | Simulate monitoring of heart rate and temperature |
| 5 | Diagnostic Records | Log and view diagnostic reports such as X-Ray and Ultrasound |
| 6 | Dosage Tracker | Record medicine dosages for patients |
| 7 | Gas Cylinder Decoder | Identify medical gases by cylinder color |
| 8 | Oxygen Level Logger | Update and monitor oxygen saturation levels |
| 9 | Patient Summary | Generate consolidated patient health summaries |
| 10 | Journey Reflection | Summary of learning outcomes from the training program |

## Running the Project

### Clone the repository

```
git clone https://github.com/Mithanya/Healthcare-Systems-Python.git
```

### Navigate to the project directory

```
cd Healthcare-Systems-Python
```

### Run any module

```
python patient_management.py
```

You can run each script individually to simulate different healthcare workflows.
